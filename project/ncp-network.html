PROJECTS

NAVER Cloud Plattform을 이용한 네트워크 구축과 보안 설정

  • 실습 의도 : ACG와 로드밸런서의 사용 및 OS 보안
  • 기        간 : 2020.07.06 ~ 2020.07.31
  • 총 인원수 : 1명
  • 실습 도구 : Putty, Putty Gen, Xmanager Enterprise 5
  • 사용 기능 : Load Balancer, Server(Public ip, Port forwarding, ACG, Private Subnet)

EXPLANATION

    서버는 CentOS 7.3, Standard-vCPU2 & 4 Memory, Zone-2, g1세대, test-acg 사양 기준으로 실습을 진행 하였습니다.
  • 두 개의 서버를 로드 밸런서를 이용해 접속하는 테스트를 진행 하였습니다.
  • Network Interface와 Private Subnet를 사용하여 허용된 서버만 연결이 가능하게 하였습니다.
  • OS 보안 작업인 root 접근 제한과 key 인증 접근을 각각 테스트 하였습니다.
  • 접속 보안 작업인 SSL VPN 접속 테스를 진행 하였습니다.
  • 콘솔 보안 작업인 2차인증 설정을 하였습니다.
