아두이노 엘레베이터

  • 기획 의도 : 아두이노와 블루투스 컨트롤을 통해 작동하는 엘레베이터
  • 기        간 : 2019.03.02 ~ 2019.06.28
  • 총 인원수 : 1
  • 개발 도구 : Arduino
  • 사용 언어 : c
  • 담당 업무 : 스크립트 개발

설명

  • 블루투스 통신을 통해 컨트롤이 가능합니다.
  • 상단에 층수를 알 수 있도록 LCD 모니터를 부착 해 두었습니다.
  • 하단에 초음파 센서를 부착하여 엘레베이터의 위치를 판단해 원하는 층 까지의 거리를 계산하여 이동시킵니다.
  • 층 마다 led를 달아 도착 시점을 알수 있습니다.
