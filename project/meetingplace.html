PROJECTS

Meeting Place

  • 기획 의도 : 수도권의 식당들이 홍보를 목적으로 쿠폰을 발급하고 게시판을 운영할 수 있게 해주는 웹 사이트 시스템
  • 기        간 : 2019.05.28 ~ 2019.06.14
  • 총 인원수 : 1명
  • 개발 도구 : eclips, mysql
  • 사용 언어 : javascript

EXPLANATION

  • 회원제 서비스를 위하여 로그인 및 회원가입 등 회원제 기능이 있습니다. 회원은 다음 기능을 사용할 수 있습니다.
    • 게시판 글 작성 및 수정
    • 쿠폰 다운로드 및 관리
    • 내 정보 관리, 비밀번호 변경 등의 계정 관리
    • 사업장 좋아요 버튼을 눌러 즐겨찾기 목록에 추가 및 삭제
  • 쿠폰 관리와 지점 정보 관리를 위하여 사업자 회원이 존재합니다. 사업자 유저는 다음 기능을 사용할 수 있습니다.
    • 쿠폰 등록 및 삭제 : 시스템에 쿠폰을 추가 또는 삭제할 수 있습니다.
    • 사업장의 정보를 등록하고 수정 할 수 있습니다.
  • 시스템의 관리와 게시판의 관리를 위해 관리자 회원이 존재합니다. 관리자는 다음 기능을 사용 할 수 있습니다.
    • 사업자의 사업장 등록을 승인 및 삭제 요청 승인
    • qna 페이지 관리
