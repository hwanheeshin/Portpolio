PROJECTS

NAVER Cloud Plattform을 이용한 서버 구축 실습

  • 실습 의도 : NCP의 서버의 생성과 관리 및 접속 방식의 이해
  • 기        간 : 2020.07.06 ~ 2020.07.31
  • 총 인원수 : 1명
  • 실습 도구 : Putty, Putty Gen
  • 사용 기능 : Server( Public ip, Port forwarding, ACG )

EXPLANATION


  • 서버 설정은 CentOS 7.3, Standard-vCPU2 & 4 Memory, Zone-2, g1세대, test-acg를 기준으로 실습을 진행 하였습니다.
  • 서버를 생성하고 공인 ip를 할당받아 ACG를 설정하는 실습을 진행 하였습니다.
  • 콘솔에서 서버의 정보를 수정하고, 그룹 별로 관리하는 방법과 인증키를 변경하는 실습을 진행하였습니다.
  • 포트 포워딩을 설정한 후, Putty를 이용해 접속하는 실습을 진행 하였습니다.
