PROJECTS

NAVER Cloud Plattform을 이용한 스토리지 구축

  • 실습 의도 : Block Storage, NAS, Object Storage의 구축
  • 기        간 : 2020.07.06 ~ 2020.07.31
  • 총 인원수 : 1명
  • 실습 도구 : Putty, Xmanager Enterprise 5, CloudBerry Explorer for Amazon S3
  • 사용 기능 : Storage(NAS, Object Storage, Snap Shot), Server(Port forwarding)

EXPLANATION

    서버는 CentOS 7.3, Standard-vCPU2 & 4 Memory, Zone-2, g1세대, test-acg 사양 기준으로 테스트를 진행 하였습니다.
  • 스토리지를 생성하고 서버에 연결 후, 파티셔닝, 마운트, fstab 등록 등의 작업을 진행 하였습니다.
  • 연결된 서버에서 스토리지를 umount 후, 다른 서버에 재 할당 하는 테스트를 진행 하였습니다.
  • 스토리지를 확장하고, 스냅샷을 생성하여 서버에 축적하는 작업을 진행 하였습니다.
  • NAS를 구축하는 작업을 진행 하였습니다.
  • Object Storage를 생성후, CloudBerry 연동 및 Fuse 서버를 연동하는 작업을 하였습니다.
